# 计算规则

## 时间

内部按整数分钟计算：

```text
有效分钟 = 结束时间 - 开始时间 - 有效休息分钟
```

结束必须晚于开始；单班默认最长 24 小时；休息必须落在班次内且不能重叠。两个班次只在半开区间 `[start, end)` 重叠时冲突，因此前一班结束恰好等于后一班开始是允许的。取消班次不参与统计，缺勤班次保留计划值但不产生实际工资。

## 工资

货币使用 `Decimal`：

```text
基本工资 = 有效分钟 × 时薪 ÷ 60
百分比加薪 = 命中分钟的基本工资 × 比例
固定时薪加薪 = 命中分钟 × 每小时固定加薪 ÷ 60
班次固定加薪 = 满足规则的固定金额（每条规则每班一次）
劳动工资 = 基本工资 + 加薪 + 奖金 - 扣减
预计总额 = 劳动工资 + 交通费
```

跨午夜班次按每分钟所在的当地日期、星期和日内时间判断规则。允许叠加的规则分别累加；同一分钟命中多条不可叠加规则时，只采用优先级最高的一条。休息分钟不会命中加薪。

当前界面支持对班次总有效分钟做 1/5/10/15/30 分钟的向下、四舍五入或向上取整。工资最后以工作设置的日元单位四舍五入。

## 时薪历史与快照

时薪以金额和生效时间保存。查询班次时选取班次开始时刻有效且生效时间最新的记录。保存班次时同时写入使用时薪、基本工资、加薪和总额快照；计划与实际统计仍可根据各自时间独立重算。

## 工时风险

所有工作合并为同一个周合计。风险阈值和上限均来自用户设置，内部单位为分钟。提示仅表述为“接近/超过设置值”，不会判断合法性。

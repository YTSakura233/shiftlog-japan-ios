常见问题

App 会自动判断我能工作多少小时吗？

不会。App 只按照你自己确认并设置的上限进行合计与提醒，不能判断你的许可是否合法或适用哪项规则。请以在留资格、资格外活动许可、学校规定及官方说明为准。

工资估算为什么和工资单不同？

估算使用你在工作资料中设置的时薪、加薪、取整、休息和交通费规则。雇主实际计算可能还包含税费、保险、节假日规则或其他项目。你可以记录工资单应发、扣除和实际到账金额进行核对。

通知或日历权限被拒绝后还能使用吗？

可以。权限被拒绝不会影响班次、工时和工资管理等核心功能。你可以之后在 iPhone 的系统设置中重新授权。

如何备份数据？

在 App 的“我的 → 数据”中导出本地 JSON 备份。你也可以导出 UTF-8 CSV 班次记录。备份文件可能包含敏感信息，请谨慎选择保存位置和分享对象。

Android 版本什么时候发布？

Android 版本已经列入正式开发计划，目前正在开发。确定发布时间后会在官网更新；现阶段不会给出尚未确认的日期。

反馈问题

请在 GitHub 项目中提交问题，并尽量说明设备型号、系统版本、App 语言、复现步骤和预期结果。请勿上传包含姓名、地址、证件号码、工资单或雇主隐私的信息。
